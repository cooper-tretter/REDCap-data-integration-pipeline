Recent improvements to integrate.py:

â€¢ Timepoint-specific questionnaire mapping: Questionnaires only appear at
  timepoints where they're administered per protocol. For example, MEQ-4,
  EBI, and CEQ only show data at 3d (dosing session), while PHQ-9 and GAD-7
  appear at baseline and follow-ups but not 3d.

â€¢ Column ordering for short/long questionnaire versions: For questionnaires
  with short and long versions (like AUDIT), columns are ordered so that
  items included in BOTH versions appear first, followed by items only in
  the long version. For example, AUDIT-C items (questions 1-3, completed by
  all participants) appear before the remaining AUDIT items (questions 4-10,
  only completed by those taking the full 10-item assessment).
